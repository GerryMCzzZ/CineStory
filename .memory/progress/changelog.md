## v0.1.0 (2025-03-11) - 初始版本

### 新增 🎉
- ✅ 项目初始化
  - Spring Boot 后端项目
  - Vue 3 前端项目
  - Docker 编排配置

- ✅ 项目记忆系统
  - 目标管理
  - 进度追踪
  - 决策记录
  - 代码自检
  - 会话记录

- ✅ 数据库设计
  - 6 张核心表
  - Flyway 版本管理

- ✅ 文本切片服务
  - 多级切片策略
  - 场景类型检测
  - 对话提取功能

- ✅ 前端架构
  - Vue Router 路由配置
  - Pinia 状态管理
  - Axios API 封装
  - WebSocket 客户端
  - 5 个页面组件

### 技术栈总结

**后端**:
- Java 17 + Spring Boot 3.x
- MySQL 8.0 + Redis + MinIO
- Spring Batch + WebSocket

**前端**:
- Vue 3 + Vite
- Vue Router 4 + Pinia
- UnoCSS + Axios

**部署**:
- Docker + Docker Compose
- Nginx 反向代理
