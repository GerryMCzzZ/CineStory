# Phase 1: MVP 目标

## 阶段目标

验证"小说转动漫视频"核心流程的可行性，交付可用的最小产品。

---

## 功能范围

### 核心功能（必须）
- [x] 小说文本上传
- [x] 智能文本切片
- [ ] LLM 提示词生成
- [ ] 单一视频源集成（Runway）
- [ ] 基础视频拼接
- [ ] 简单进度展示

### 支撑功能
- [x] 数据库设计
- [x] 用户认证（基础）
- [ ] 文件存储（MinIO）
- [ ] 错误处理
- [ ] 日志记录

### 暂不实现
- ~~多视频源~~
- ~~高级转场效果~~
- ~~AI 配音~~
- ~~角色一致性~~
- ~~前端美化~~

---

## 技术债务暂缓

以下技术优化将在 Phase 2 处理：
1. 单元测试覆盖
2. API 性能优化
3. 缓存策略
4. 消息队列优化

---

## 验收标准

### 功能验收
- [ ] 能成功处理 5000 字小说
- [ ] 生成至少 10 个视频片段
- [ ] 成功拼接成完整视频
- [ ] 端到端流程无报错

### 性能验收
- [ ] 文本切片 < 5 秒
- [ ] 提示词生成 < 30 秒
- [ ] 单个视频生成 < 5 分钟
